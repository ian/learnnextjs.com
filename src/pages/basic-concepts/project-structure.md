## Directory Structure

While the choice of how to organize your project is ultimately up to you, developers tend to follow this project structure in general.

```
./myapp
│
├── .next/
├── components/
├── pages/
│   ├── _app.tsx
│   └── _document.tsx
├── public/
├── package.json
├── next.config.js
├── tsconfig.json
└── yarn.lock
```
